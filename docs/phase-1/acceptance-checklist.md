# DropRoom 一期验收清单

本清单对应一期 PRD 第12节。自动化测试只能证明协议和状态机，真实浏览器、网络和生产代理仍需要人工验收。

| PRD 验收项                          | 当前证据                         | 状态           |
| ----------------------------------- | -------------------------------- | -------------- |
| 无账号创建并通过码或链接加入        | API 主流程、Web 邀请加入回归测试 | 自动化通过     |
| 最多9名成员查看实时内容             | 9人边界测试、SSE 快照测试        | 自动化通过     |
| 同名成员使用数字 ID 区分            | RoomStore 成员快照测试           | 自动化通过     |
| 新成员加载文字和文件历史            | RoomStore 与 API 历史测试        | 自动化通过     |
| 传输文字、普通文件和图片            | API 上传下载、长文本转文件测试   | 自动化通过     |
| 单批300 MB、房间1 GB和实例30 GB限制 | 容量边界与并发预占测试           | 自动化通过     |
| 上传进度、续传、容量和到期提醒      | 分片续传测试、Web 进度与恢复测试 | 待实机体验验收 |
| 发送设备离线后仍可下载              | RoomStore 文件持久读取测试       | 自动化通过     |
| 房主转移、解散和自动销毁            | RoomStore 状态机与下载保留测试   | 自动化通过     |
| 房间销毁后的内容清理                | 即时清理与下载中30分钟清理测试   | 自动化通过     |
| 成员凭证不进入 URL 或本地存储       | Cookie 鉴权、旧缓存迁移测试      | 自动化通过     |

## 发布前人工验收

- [ ] 按 [浏览器兼容性与验收](./browser-compatibility.md) 完成桌面端和移动端实机记录。
- [ ] 在真实 HTTPS 域名验证创建、加入、SSE、上传、预览和下载。
- [ ] 验证接近300 MB的文件可以通过多个分片穿过最外层反向代理。
- [ ] 验证断网、切网、移动端后台和 EventSource 自动重连。
- [ ] 验证桌面端和移动端粘贴截图、图片及普通文件可以上传；长纯文字先进入输入框且可编辑，点击发送后按阈值转为 TXT 并计入容量。
- [ ] 验证 TXT 文件卡片可以打开全文预览，并可复制全部、下载和关闭弹窗。
- [ ] 验证超高文字气泡限制在256px内，鼠标、触屏及键盘均可在气泡内部滚动。
- [ ] 验证刷新后重新选择同一个文件可以继续上传。
- [ ] 验证30分钟提醒、5分钟强提醒和最终销毁。
- [ ] 验证浏览器通知的授权、后台提醒、关闭开关，以及 Windows/Linux 与 macOS 的输入框全选快捷键。
- [ ] 验证上传卡片位置固定、Header 两小时阈值无抖动，以及解散时下载中资源最多保留30分钟。
- [x] 验证 SSE、预览和下载 URL 不包含成员 token。
- [ ] 验证健康检查、磁盘、5xx、重启和 Redis 告警。
